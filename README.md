Welcome to the **CodeFluxz** official website repository. CodeFluxz is an innovative tech agency focused on building the trust and opportunity layer for the AI-powered future. We design, architect, and deploy high-performance applications like **Enimi AI**, **Potaketch**, and **Famtree**.

## 🚀 Features

- **Cinematic Scroll Experience:** Butter-smooth, hardware-accelerated animations using GSAP and ScrollTrigger.
- **Physics-Based Smooth Scrolling:** Integrated Lenis scroll physics for a premium, agency-grade feel.
- **Dynamic Product Showcases:** Interactive vertical and horizontal scrolling interfaces to present our core ecosystem.
- **Modern UI/UX:** Built with a dark, agency-inspired aesthetic, utilizing glassmorphism, precise typography, and fluid hover states.
- **Responsive Design:** Fully optimized for fluid scaling across mobile, tablet, and desktop environments.

## 🛠️ Tech Stack

- **Core:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation Engine:** [GSAP (GreenSock)](https://greensock.com/gsap/) + [Framer Motion](https://www.framer.com/motion/)
- **Scroll Physics:** [Lenis](https://lenis.darkroom.engineering/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 💻 Run Locally

To get a local copy up and running, follow these simple steps.

**Prerequisites:** Node.js (v18+ recommended)

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/your-username/codefluxz.git
   cd codefluxz
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open your browser and navigate to `http://localhost:5173`
